Frequently Asked Questions

How often are new promo codes released?

New codes are typically released alongside major game updates, special events, holidays, and content creator partnerships. There's no fixed schedule, but active players can expect several codes per month.

Can I use the same code multiple times?

Most promo codes are one-time use per account. Once redeemed, the same code cannot be used again on that account. Codes are also typically single-account — you cannot redeem the same code on multiple accounts.

What happens if I enter a wrong code?

The game will display an error message indicating the code is invalid or expired. Double-check for typos, case sensitivity, and ensure the code hasn't expired. Expired codes cannot be redeemed.

Do codes expire if I don't use them?

Yes. Most codes have a limited redemption window, typically 7 to 30 days from their release date. Some special event codes may have shorter windows. Always redeem codes as soon as you find them.
