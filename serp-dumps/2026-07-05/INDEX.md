# SERP Dump Index — 2026-07-05

## Purpose
Historical snapshot of Google Search Engine Results Page positions for tracked queries. New dumps are created periodically (weekly recommended) to track ranking changes over time.

## Directory Structure
```
serp-dumps/
  YYYY-MM-DD/                  # Date of dump
    INDEX.md                   # This index file
    queries/                   # One file per query
      gem-calculator.txt
      promo-codes.txt
      free-gems.txt
      ...
```

## How to Create a New Dump
1. Copy the previous date's directory as a template
2. Update each query file with fresh search results
3. Update this INDEX.md with the new date
4. Commit
