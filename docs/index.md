# Documentation

## Overview

Gem rewards infographic for Invincible Mobile Game featuring interactive charts and visualizations.

## Gem Data

| Category | Gems | Description |
|----------|-----|-------------|
| Season Rewards | 1,820 | Elite League I (810), Invincible League (560), Elite League II (450) |
| Event Rewards | 500 | The Long Haul (300), Earth's Defenders (200) |
| Warfare Rewards | 750 | 5 battles per 2 weeks |
| Daily/Weekly Rewards | 1,100 | Daily login (700) + Weekly bonus (400) |
| Promo Code | 300 | Code: 30KGTG |

**Total: 3,920 gems** (one-time + recurring monthly)

## Charts

- **Distribution**: Doughnut chart showing gem breakdown by category
- **Performance**: Radar chart showing player performance metrics
- **Top Rewards**: Bar chart of individual reward amounts
- **Monthly**: Line chart showing monthly income over time

## Tech Stack

- Tailwind CSS for styling
- Chart.js for interactive charts
- Font Awesome for icons
- Rajdhani font for typography
